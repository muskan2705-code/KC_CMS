# KC CMS

Strapi CMS for the KC Fixit website.

## Development

Start the Strapi application with auto-reload enabled:

```bash
npm run develop
```

Admin panel:

```text
http://localhost:1337/admin
```

## Production

Build the admin panel:

```bash
npm run build
```

Start Strapi without auto-reload:

```bash
npm run start
```
